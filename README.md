# Feedback Collector

## Description
Feedback Collector est un projet léger qui permet aux utilisateurs de soumettre leurs retours via un formulaire. Les réponses sont stockées et peuvent être analysées pour améliorer les services ou produits.

## Fonctionnalités
- Formulaire de feedback simple à remplir
- Collecte des réponses dans une base de données
- Tableau de bord pour visualiser les retours
- Exportation des données en CSV

## Installation
1. Clonez le dépôt : `git clone https://github.com/votre-utilisateur/feedback-collector`
2. Installez les dépendances : `npm install`
3. Démarrez le serveur : `npm start`

## Utilisation
Accédez à l'application via `http://localhost:3000` et commencez à recueillir des retours utilisateurs !
